# GOAL

> Pointer file. Canonical sources below — do not duplicate their content here.

**What:** Hybrid — Bengali-first, mobile-first multi-tenant commerce SaaS ("Shopify for Bangladesh").
Each seller gets an admin backend, a themed storefront on a subdomain (later custom domain), and
native bKash/Nagad/COD + Bangladesh courier (Steadfast/Pathao/RedX/Paperfly) integration. Hard
tenant isolation via Postgres RLS.

**Why:** Existing tools (Shopify etc.) don't natively serve Bangladesh — no bKash/Nagad/COD-first
checkout, no local courier wiring, no Bengali-first mobile UX. Hybrid is built around those.

**For whom:** Bangladeshi online sellers (F-commerce → growing SMB merchants).
